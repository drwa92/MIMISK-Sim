Realistic Reef Pack for Unity

Files:
- realistic_reef_pack.obj
- realistic_reef_pack.mtl
- healthy_coral_texture.png
- bleached_coral_texture.png
- rock_texture.png
- seagrass_texture.png
- sand_texture.png

How to import:
1. Copy this whole folder into:
   simulation_minirov_labust/Assets/Environment/Models/
2. Open Unity and wait for import.
3. Drag realistic_reef_pack.obj into your pool scene.
4. If materials do not load, create Unity materials manually and assign the PNG textures to Albedo/Base Map.
5. Recommended: do NOT add Mesh Collider to the entire reef. Use simple Box Colliders only on large rocks if needed.
